融资项目发起人服务协议

微风投网络平台（网址:http://www.weifengtou.com）是河南梧桐盛嘉投资管理有限公司旗下经营的网站。本协议双方为河南梧桐盛嘉投资管理有限公司（简称“微风投网络平台”）与通过微风投网络平台审核通过的实名融资项目发起人（简称“您”或“融资方”）。在您使用微风投网络平台提供的服务之前，请务必认真阅读本协议的全部内容。本协议一经您点击确认并同意接受，即在您与微风投商业模式之间产生法律效力。

第一条 定义与解释

1.1 项目发起人：在微风投网络平台注册并发起融资项目的单位或个人。

1.2 项目投资人：在微风投网络平台注册并通过微风投商业模式进行投资的单位或个人。

1.3 费用：在微风投网络平台注册的用户（含融资方、投资方以及其他用户）不需缴纳任何费用，但是融资方因为项目推介、宣传、展示以及线下路演等活动的费用根据实际情况，由融资方和投资方以及平台方协商处理。

1.4 居间服务费：为保证微风投商业模式投资方的资金安全，融资项目项下资金均通过第三方管理公司代为划转，由此产生的支付手续费由项目发起人负担，项目投融资对接成功后，第三方管理公司将收取相应的居间费用，作为项目调查、风险评估、收益预测等的手续费用。

第二条 本协议的构成及效力

2.1 所有微风投网络平台已经发布的或将来发布的各类规则，作为本协议不可分割的一部分，与本协议具有同等的法律效力。

2.2 微风投网络平台有权根据需要适时修订本协议及各类规则，并在网站上公示。变更后的协议和规则一经在微风投网络平台上公示，即发生法律效力。

2.3 在使用微风投网络平台提供的服务之前，请务必认真阅读本协议的全部内容。如您对本协议有任何疑问，应向微风投网络平台400-700-8998咨询。本协议生效后，您不应以未阅读或不接受本协议的内容为由，主张本协议无效或要求撤销本协议。

2.4 您应该按照本协议约定行使权利并履行义务。如您不能接受本协议的约定，包括但不限于不能接受修订后的协议及各类规则，则您应立即停止使用微风投网络平台针对项目发起人提供的服务。如您继续使用微风投网络平台针对项目发起人提供的服务，则表示您同意并接受本协议及各类规则的约束。

第三条 项目发起人的资格

3.1 作为项目发起人，您须为经过微风投网络平台审核通过的实名注册用户，完全理解并接受本协议。作为拟融资项目的单位应为依法成立并登记备案的企业法人、经济实体或其他组织。

3.2 作为项目发起人的个人，应为年满18周岁并具有完全民事行为能力和民事权利能力的自然人，如项目发起人未满18周岁，应由其监护人代为履行本协议权利和义务。

3.3 您应按照微风投网络平台的要求，进行必要的身份认证包括但不限于身份证、护照、学历证明等的认证；以及主体资质认证包括但不限于法人营业执照、税务登记证、组织机构代码证、企业征信机构代码证等。

3.4 您应拥有在中国大陆地区开户并接收人民币结算的银行卡或对公结算账户。

3.5 您应妥善保管在微风投的用户名和密码，凡使用您的用户名和密码登陆微风投网络平台进行的一切操作，均视为您本人的行为，一切责任由您本人承担。

第四条 拟融资项目应符合以下要求

4.1 您在微风投网络平台上发起的项目，应明确具体内容如下:

  （1）项目所属的行业；

  （2）项目所属公司的成立时间以及经营情况；

  （3）项目的投资总额，其中自筹部分和拟融资部分的金额；

  （4）项目未来市场营销计划和预期收益的预测；

  （5）融资金额偿还本金和支付投资收益的计划；

  （6）其他与项目有关的情况。

4.2 您发起的项目不得包含非法、色情、淫秽、暴力等内容，不得含有攻击性、侮辱性言论，不得含有违反国家法律法规、政策的内容及其他微风投网络平台认为不适宜的内容。

4.3 您在微风投网络平台发起的项目，不得在国内外同类网站同时发起。

4.4 您应对自己及您发起的项目进行介绍，内容包括“关于我”、“发起该项目的目的”、“项目的进展和风险”、“为什么选择微风投商业模式发起该项目”、“融资项目运营成功后的预期回报”等。同时，您应向项目投资人充分说明项目存在的风险及挑战，以便于项目投资人对项目有全面充分的了解，从而独立慎重作出是否投资的决定。

4.5 您发起的项目最好属于创新型、成长型和科技型，具有较高的发展潜力和市场成长性和可行性。

4.6 您发起的项目不应与第三方存在任何权利纠纷，否则因此导致的一切损失（包括微风投网络平台因此被第三方权利追索而遭受的一切损失）由您本人承担，与微风投网络平台以及河南梧桐盛嘉投资管理有限公司无关。

4.7 在项目发起后的运行过程中，您应及时回复其他用户的提问，与项目投资人进行充分互动，促成项目推介展示对接成功。

4.8 项目发起后，您应及时更新项目进展情况，包括但不限于以照片、视频、素描等方式展示项目的进度信息。

第五条 项目回报

5.1 您在发起项目时，应明确项目融资成功后的预期回报时间。

5.2 如您与项目投资人在微风投合作协议中约定的偿还本金和支付投资收益的过程中发生纠纷，一切责任由您本人承担，如因此给微风投以及关联机构造成经济或名誉损失，您应赔偿损失、恢复名誉。

5.3 如您在偿还本金和支付投资收益的回报过程中，与第三方发生纠纷，一切责任由您本人承担，与微风投网络平台以及关联公司无关。

第六条 协议终止

6.1 以下任一情形出现，本协议即终止:

  （1）您发起的项目违反了国家法律法规、政策或微风投网络平台使用规则或本协议约定，微风投网络平台停止为您提供服务的；

  （2）您本人不同意接受本协议约定（含微风投网络平台发布的各类规则），并停止使用微风投网络平台针对项目发起人提供的服务的；

  （3）您不符合本协议约定的项目发起人应具备的资格的；

  （4）其他原因导致本协议终止的。

6.2 本协议终止后，不影响您因在本协议终止前的行为应承担的义务和责任。
