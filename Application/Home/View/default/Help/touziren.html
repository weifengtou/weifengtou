第一条 定义与解释

1.1 项目发起人：在微风投网络平台注册并发起融资项目的单位或个人。

1.2 项目投资人：在微风投网络平台注册并通过微风投商业模式进行投资的单位或个人。

1.3 费用：在微风投网络平台注册的用户（含融资方、投资方以及其他用户）不需缴纳任何费用，但是融资方因为项目推介、宣传、展示以及线下路演等活动的费用根据实际情况，由融资方和投资方以及平台方协商处理。

1.4 资金支付手续费：为保证微风投商业模式投资方的资金安全，融资项目项下资金均通过第三方支付公司代为划转，由此产生的支付手续费由项目发起人负担，项目投融资对接成功后，第三方管理公司将收取0.2%的居间费用，作为项目调查、风险评估、收益预测等的手续费用。

第三条 风险提示

使用微风投网络平台展示、推介的项目，以及从微风投网络平台中获取其他信息进行风险评估和投资决策时需关注以下风险：

3.1 宏观经济风险：由于我国及周边国家、地区宏观经济环境和周边市场的变化，可能会导致在初创期间的小微企业的经营存在较大的不稳定性，从而导致您拟投的项目或者已投的项目的价值可能会有不同程度的贬损，因此可能影响您的投资收益。

3.2 政策风险：微风投商业合作模式所推荐的小微企业所处行业的法律、法规及相关政策发生变化，或者微风投网络平台的制度、规则发生变化，可能引起您拟投资和已投资小微企业的经营业绩和披露信息的波动，因此可能影响您的投资收益。

3.3 公司运作风险：由于参与微风投商业合作模式的项目融资方的内部管理和外部决策不在微风投网络平台以及关联公司的控制范围内，微风投商业合作模式只能在法律法规及相关业务规则允许的范围内尽可能向您进行完整的信息披露，鉴于此，您拟投资和已投资的项目主体业务不确定性造成的风险可能会影响到您的投资收益。另，微风投网络平台所推介展示的项目处于种子期或初创期，规模较小，抗市场风险和行业风险的能力较弱，主营业务收入可能波动较大，在一定程度上会影响到你的投资收益。

3.4 技术风险：由于微风投商业合作模式中的项目资料信息是通过网络技术来实现的，微风投网络平台会尽力保障平台正常运行，但这些技术存在被网络黑客和计算机病毒攻击的可能，可能会导致您拟投或者已投的项目资料失真，导致您的判断失误和决策失误。

3.5 不可抗力因素导致的风险：诸如地震、火灾、水灾、战争等不可抗力因素可能导致微风投网络平台故障，也可能导致您的投资决策出现偏差导致投资损失。

3.6 其他微风投商业合作模式无法预见、无法避免或无法控制的风险。
